This directory is the **single source of truth** for AI context on the
`dspeech` project. Agents working on this project MUST:

1. Read `current-context.md` first for the rolling summary.
2. Read `docs/adr/` for binding architectural decisions.
3. Read `.ai/project-state.md` for live runtime state.
4. Only then consult Notion (read-only mirror) or any chat history.

## Layout

- `README.md` (this file) - layout + entry rules.
- `current-context.md` - rolling 1-page current context. Updated by the
  `knowledge-curator` role after every substantive run.
- `docs/adr` - architectural decision records, append-only.
- `.ai/project-state.md` - runtime project state pointer.

## Task prefix

All chat messages and Notion items scoped to this project use the prefix
`[Dspeech]`. Untagged messages are NOT this project.

## Source-of-truth ordering

1. This repo + this KB - authoritative.
2. Notion database - read-only mirror.
3. Telegram / Open WebUI / Obsidian - UX surfaces only.

Conflicts resolve in favour of layer 1.
